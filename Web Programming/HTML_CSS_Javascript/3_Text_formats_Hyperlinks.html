Importing a hyperlink

  • If you have another .html file in the same folder (project), and you want to link it in this page, you can add its filename in the string value of "href".

  • You, also, use URL's you can open via this HTML file, like the upper link it will open on the same page (if target is set on "_self" -- this is the default value).

  • On the other hand, you can open that link on a new page (if target is set on "_blank" instead of "_self") it will open on the same page.

  • Also, check how a title/ description is showing over this link (if you add a string value in the title field) when you hover mouse over the hightlighted area.

Contact mail + auto open mailbox
