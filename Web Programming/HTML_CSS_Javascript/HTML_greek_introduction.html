Kανόνες και αναγνώριση λαθών:

  • Όταν κάτι είναι λάθος στην HTML, απλώς το αγνοεί, και συνεχίζει με τα υπόλοιπα.
  • Αν έχουν γραφεί πολλαπλά κενά ή ακόμη και πολλαπλά new line, η HTML τα αγνοεί και τα αντικαταστεί με ένα μόνο κενό χαρακτήρα.

Δομή της HTML

  1. Στην αρχή κάθε εγγράφου είναι αναγκαίο αρχίσω γράφοντας <!doctype html>
  2. Η βασική κι απαράλλακτη δομή κάθε αρχείου HTML είναι:

        <!doctype html>
        <html>
            <head>
            ...
            </head>
           <body>
            ...
            </body>
        </html>

  3. Οι περισσότεροι "τύποι εντολών/ κειμένου" (elements) είναι μορφής <element> ... </element>.
    Υπάρχουν όμως και elements που είναι μορφής <element>, τα οποία δεν έχουν κάποιο περιεχόμενο να περικλύσουν.

Κάποια HTML elements:

  • Άλλο το element <tittle>, το οποίο δείχνει το τι θα φαίνεται ως όνομα καρτέλας στο browser, και...
  • ...άλλο τα header, δηλάδη τα <h1>, <h2>, ..., <h6> που είναι κείμενο ευδιάκριτο και μορφοποιημένο να τραβάει τη προσοχή κάποιου, ως τίλος κειμένου/ων

Σχετικά με λειτουργίες της HTML

  • Για να γράφω (και) σε ελληνικά, βάζω στο head το < meta charset="utf-8" >
  • Με βοήθεια CSS, μπορώ να τροποιπήσω το style του κειμένου, σε μορφή (αν είμαστε σε παράγραφο) < p style= "..."> ..text.. </p>
  • Για πολλαπλές απαιτήσεις στο style, χωρίζω κάθε μια με ερωτηματικό, όλες εντός των ίδιων εισαγωγικών
  • Μπορώ να συμπεριλάβω hyperlink, γράφωντας < a href= "url">..text over url..</a>
  • Μπορώ να κάνω elements τύπου ordered list (ol) ή unordered list (ul), εσωτερικά και των δύο, κάθε αντικείμενο δηλώνεται με το <li> ... </li>. Αν θέλω η αρίθμηση να γίνεται αντίστροφα (Ν ως 1), τότε μετά το τύπο λίστας γράφω τη παράμετρο reversed.

Σχετικά με (επιλέον) κενά διαστήματα:

  • Για αλλαγή γραμμής, χρησιμοποιώ το <br>
  • Για σχεδίαση διαχωριστικής γραμμής κατά μήκος της οθόνης, χρησιμοποιώ το <hr>

Για τη μορφοποίηση κειμένου:

  • Για να μορφοποιήσω το κείμενο:
    • BOLD: γράφωντας <b> ...text... </b>
    • ITALICS: γράφωντας <em> ...text... </em>
    • UNDERSCORE: γράφωντας <u> ...text... </u>
  • Για να βάλω χρώμα στο κείμενο, (πχ σε One-line command, για paragraph) γράφω:
    1. με κωδικό RGB, όπως < p style=color: rgb(255, 165, 0)">
    2. ή, με κάποια σταθερά (τριπλέτα RGB) που προυπάρχει, όπως < p style=color: orange">
  • Για να ελέγξω τη στοίχιση του κειμένου (πχ σε One-line command, για paragraph):
    • Centrer: γράφωντας < p style= "text-align: center">..text..</p>
    • Left: γράφωντας < p style= "text-align: left">..text..</p>
    • Right: γράφωντας < p style= "text-align: right">..text..</p>
  • Για να τυπώσεις ειδικούς χαρακτήρες, γράφεις &# ακολουθούμενο ένα κωδικό αριθμό (πχ, αν μετά το &# γράψω το 60, τυπωνει το < ).
    Δες κι άλλους χρήσιμους κωδικούς εδώ.

Εισαγωγές που μπορώ να κάνω:

  • Εισαγωγή εικόνας
    Χρησιμοποιώ το όνομα της (αν είναι στον ίδιο φάκελο) ή το path (αν είναι σε υποφάκελο εντός του φακέλου που περικλύειται ο κώδικας μου).
    Η παρακάτω εικόνα εισήχθει ως <img src="/assets/flowers.jpg" alt="An image of flowers" width="200" height="150">.
    Μάλιστα, προκειμένου να τοποθετηθεί στο κέντρο, η άνωθεν εντολή μπήκε εντός της
    <div style="text-align: center"> ..εντολή.. </div>
