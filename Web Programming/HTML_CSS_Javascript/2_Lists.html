Unordered list

List with no numbering of elements. Shows them with bullets (by default). List symbolided with the <ul> tag, and each list item inside is in-between <li> tag.

List with no numbering of items

  • A list item.
  • Another list item.
  • And another list in list.

Nested list

This works with other types of lists, too. Just put a whole structure of list inside another -- add it in a <li> tag.

  • A list item.
  • Another list item.
    • A nested item in a list.
    • Another nested item in the list.
  • And another list in list.

Ordered list

List with its elements having a number in front of it. List symbolided with the <ol> tag, and each list item inside is in-between <li> tag (same as unordered list).

List starting counting: 1 (default)

  1. First item
  2. Second item
  3. Third item

List starting counting: 7 (with attribute start="num")

  1. First item
  2. Second item
  3. Third item

List starting counting: A (with attribute type="A")

  1. First item
  2. Second item
  3. Third item

Description list

List with pairs - one is a term (word to be "defined") and the other is an explanation (the desctiption/ definition). List symbolided with the <dl> tag, and each list-key (term to be defined) is symbolized with <dt>, and its defintion (the description of term) with <dd> tag.

List with term descriptions

Key word #1

Another term

Andddd another term
Here we define the last term of our list.
