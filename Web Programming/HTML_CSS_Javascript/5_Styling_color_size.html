Coloring

Add base colors on style - head

It is encouraged to apply base page styling (for each tag set alignment, border size, padding, etc or even width, etch)
in the begging of the HTML document.
Do so by using the <style> element on the top (somewhere inside the <head>).

In-line style command

You can also add specific styling for specific elements, by adding the style=... after the tag key-word.
If the element has a general instruction (from style in head section) but later it gets an in-line style command,
then it keep the in-line (more specific/ exception) styling). Exaple:
