Label fields

Text to explain information wanted in fields.

  • <label for="codeNameLabel" > LabelNameSOnSite </label>

Input fields

Self closing argument for a variety of input fields.

  • For text input
    • <input type="text" id="fname" name="fname">
    • <input type="text" id="fname" name="fname" placeholder="Jon">
    • <input type="text" id="fname" name="fname" placeholder="Jon" required>
  • For numerical input
    • <input type="number" id="age" name="age">
    • <input type="number" id="age" name="age" required>
    • <input type="number" id="age" name="age" min="18" required>
  • For date
    • <input type="date" id="Date" name="Date">
    • <input type="date" id="Date" name="Date" required>
  • For checkbox
    • <input type="checkbox" id="checkRule" name="checkRule">
    • <input type="checkbox" id="checkRule" name="checkRule" required>
  • For file uploading
    • <input type="file" id="pdfFile" name="pdfFile" accept=".pdf">
    • <input type="file" id="multiType" name="multiType" accept=".pdf, .jpg, .png">
  • For reset button
    • <input type="reset" id="Reset" value="Reset">
  • For reset button
    • <input type="submit" id="Submit" value="Submit">

Select from menu

Pick one from a drop-down menu.
Define the menu as:

  • <select id="aMenu" name="aMenu"> ... </select>

And, in the ..., add each option available for user to pick.

  • <option value="opt1"> Text for option 1 </option>
  • <option value="opt2"> Text for option 2 </option>
