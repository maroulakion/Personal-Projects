Tables

Create default table styling with CSS

It is strongly encouraged to set some default styling for table in a <style> element on the top (somewhere inside the <head>). For exaple, it is encouraged to set alignment, border size, padding, etc or even width.
In this HTML file, CSS is applied that way on <table> and <th, td> elements.

Simple table - 2 rows (no styling)

A table in HTML is basically many rows one of top of the other. So when creating one, you need to fill it row by row.
First, each row is included in <tr> (table row) tag.
Inside, we can add <th> (table header) elements if the cell works like title,
or <td> (table data) if the cell holds actual information.
