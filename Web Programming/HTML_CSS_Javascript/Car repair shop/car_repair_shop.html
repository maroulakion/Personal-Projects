Το "Συνεργείο Αυτοκινήτων - ο Μπάμπης" ιδρύθηκε από τον ομώνυμο ιδιωκτήτη το 1987, και εκ τότε συνεχίζει να καλύπτει τις ανάγκες δεκάδων πελατών καθημερινά.
Τοποθετημένο στο κέντρος της Θεσσαλονίκης (Ουτοπίας 99) και καταρτισμένο με έμπειρους τεχνικούς, μηχανικούς και αυτοκινητο-γνώμονες, είμαι εδώ για να αντιμετωπίσουμε κάθε είδος απαίτηση, ζημίας, ή αναβάθμισης επιθυμείτε!
